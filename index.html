API Integration

Dokumentasi ini menjelaskan cara menggunakan endpoint Flora.AI untuk mendeteksi penyakit tanaman melalui aplikasi pihak ketiga (Android/iOS/Web).

// Endpoint URL

POST https://domain-kamu.vercel.app/api/analyze

// Headers

Content-Type: application/json

// Request Body

{
  "image": "base64_string_tanpa_prefix",
  "mime": "image/jpeg",
  "plantName": "Tomat" // Opsional
}

Requirement

Membutuhkan `GEMINI_API_KEY` yang diset di Environment Variables Vercel. Pastikan model yang digunakan di server adalah `gemini-2.5-flash`.

Response Format

Server akan mengembalikan status 200 OK dengan format JSON berisi atribut `analysis` yang memuat teks laporan dengan format Markdown.

Tentang Flora.AI

Flora.AI adalah platform diagnostik agritech inovatif yang dibangun untuk memecahkan masalah identifikasi penyakit tanaman secara instan.

Dengan memadukan kecepatan Google Gemini 2.5 Flash dan UI yang sangat dioptimalkan, platform ini bertujuan memberikan akses layaknya pakar botani profesional langsung ke tangan petani, penghobi tanaman, dan pelajar di seluruh Indonesia.
